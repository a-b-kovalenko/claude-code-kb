# Нотатки про команди (`content/commands/`)

Окрема нотатка для команди створюється коли команда має нетривіальні параметри, режими або потребує прикладів. Прості команди — рядок у `Commands_Reference.md` без окремого файлу.

Структура нотатки команди:

1. Backlink: `[⬅️](../Commands_Reference.md)` (не до MOC)
2. `## 📝 TL;DR`
3. `## Синтаксис` — з code block `text`
4. Тематичні розділи (режими, параметри, приклади тощо)

Реєстрація:

- Рядок у відповідній таблиці `Commands_Reference.md` з посиланням на нотатку
