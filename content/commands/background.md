## 📝 TL;DR

`/background` відʼєднує поточну сесію від термінала і продовжує її у фоні. [`claude agents`](../Agents_View.md) — окремий CLI-інтерфейс (Agent View) для моніторингу і керування всіма фоновими сесіями. Разом дозволяють паралельно вести кілька задач без блокування термінала.

## Синтаксис

```text
/background [останній prompt перед відʼєднанням]
```

Аліас: `/bg`. Термінал звільняється одразу — сесія продовжує виконання у фоні.

```bash
claude agents               # відкрити Agent View
claude agents --cwd ~/proj  # тільки сесії цього проєкту
```

## Agent View

`claude agents` — повноекранний dashboard у терміналі. Сесії згруповані за станом:

- **Needs input** — чекає на відповідь
- **Working** — виконує задачу
- **Completed** — завершила

Іконки стану рядка:

| Іконка | Що означає |
| :----- | :--------- |
| `✻` / `✽` | Процес живий, відповідає одразу |
| `∙` | Процес завершився, але сесію можна продовжити |
| `✢` | `/loop`-сесія спить між ітераціями |

## Керування в Agent View

| Дія | Клавіша |
| :--- | :--- |
| Переглянути останній output | `Space` (peek panel) |
| Відповісти не виходячи | набрати у peek panel + `Enter` |
| Зайти у повну сесію | `Enter` або `→` |
| Повернутись до таблиці | `←` на порожньому промпті |
| Відʼєднатись без зупинки | `/exit` або `←` |
| Зупинити сесію | `/stop` |
| Диспатч + одразу зайти | `Shift+Enter` |

## Ізоляція файлів

Перед першим записом фонова сесія автоматично переміщається у ізольований git worktree під `.claude/worktrees/`. Паралельні сесії читають той самий checkout, але кожна пише у свій. Вимкнути: `worktree.bgIsolation: "none"` у `.claude/settings.json`.

## Воркфлоу

Існуючу інтерактивну сесію можна відправити у фон двома способами:

- `/bg` або `/background` у сесії
- `←` на порожньому промпті → відкриється Agent View

Знайти фонову сесію пізніше: `/resume` — вони позначені міткою `bg` у picker-і.

## Коли використовувати

Запустив `/batch` або довгу задачу → `/background` → перейшов до іншого → повернувся через `claude agents` коли сесія потребує відповіді.
